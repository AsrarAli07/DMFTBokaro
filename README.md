# DMFT Bokaro – Android Application

## Overview
DMFT Bokaro is an Android application developed to manage and deliver public service information efficiently.  
The app focuses on reliability, performance, and clean user experience.

## Features
- Secure authentication
- REST API integration
- Offline data support
- Clean Material UI
- Performance optimization

## Tech Stack
Java, Android SDK, REST APIs, Firebase, SQLite, Material Design

## Architecture
- MVVM pattern
- Separation of concerns
- Reusable components
